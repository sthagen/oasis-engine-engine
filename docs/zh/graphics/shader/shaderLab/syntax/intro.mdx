## 语法标准

`ShaderLab` 语法骨架如下:

```glsl
Shader "ShaderName" {
  ...
  SubShader "SubShaderName" {
    ...
    Pass "PassName" {
      ...
    }
    ...
  }
  ...
}
```

主要包含 [Shader](/docs/graphics/shader/shaderLab/syntax/shader/)，[SubShader](/docs/graphics/shader/shaderLab/syntax/subShader/) 和 [ShaderPass](/docs/graphics/shader/shaderLab/syntax/pass/) 模块。
